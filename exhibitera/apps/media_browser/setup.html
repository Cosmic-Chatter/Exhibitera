Welcome to Media Browser!

Media Browser displays collections of audio, images, and videos for users to explore. They can browse the collection, filter it based on attributes you choose, and view more information about a selected item.

To create a Media Browser, you will need:

  • The media files you want to display;
  • The media titles, captions, and any credit information you want users to see.
Definition name must not be blank.

What languages will you include?

Media Browser supports browsing in multiple languages. Select the langauges you want to include:

You must add at least one language.

How do you want to add your media?

Now, you'll add your media files. You can chooes your files now and add their details (title, caption, credit, etc.) after completing Guided Setup. Or, you can use a spreadsheet to bulk import everything right now.

Upload and select your files

Use the button below to upload and select the files you want in your browser. Audio, image, and video files in most formats are supported. You can always add more files later.

Organize your media and details

Based on the languages and details you have selected, a template has been generated. Download the template and edit it using software such as Microsoft Excel, Google Sheets, Apple Numbers, or LibreOffice Calc.

Each row of the spreadsheet represents one file in your media browser. Make sure to save the file in the CSV (comma-separated values) format.

You must select an uploaded spreadsheet listing your media.

Add your media

You must upload a media file matching each row of your spreadsheet. Media files should be images or videos in a supported file format (see Help for more infomation.)

Your spreadsheet must contain a column named Media filename.
