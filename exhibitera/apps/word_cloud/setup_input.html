Welcome to Word Cloud Input!

Word Cloud Input prompts the user to give a short answer to a question you pose. Those answers will be collected by Word Cloud Viewer and turned into a beautiful word cloud.

To create a Word Cloud Input, you will need:

  • The text of the question you want to ask;
  • The name of the collection to which you are adding responses.
Definition name must not be blank.

What question will you ask?

For best results, the question should prompt simple (ideally one word) answers.

Which collection is this for?

Responses from this input will go into a collection. To generate the word cloud, you will connect a Word Cloud Viewer to the same collection. Multiple inputs (i.e., kiosks) can all contribute to same collection and be combined into the same word cloud.

Collection must not be blank.
