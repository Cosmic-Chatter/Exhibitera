Create a new definition

Welcome to InfoStation!

InfoStation enables you to create rich, interactive digital labels. Combine text, images, and videos across one or more topics to provide more content than a printed label.

To create an InfoStation, you will need:

  • A list of languages you want to support;
  • A list of topics you want to include.
Definition name must not be blank.

What languages will you include?

Survey Kiosk enables you to present the survey in multiple langages. Select the langauges you want to include. The first language you select will be the default one (you can change this later).

You must add at least one language.

Do you want a header?

You can add a header at the top of InfoStation so guests know what the label is about. Add the header in your default language (you'll add translations later)

What topics will you include?

Each topic will became a tab in InfoStation. Add topics below in your default language (you'll add translations later).

You must add at least one topic.
Topics cannot be blank.

Choose your layout

InfoStation supports displays in either a portrait (vertical) or landscape (horizontal) layout.
