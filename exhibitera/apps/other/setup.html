Custom App Configuration

Custom App setup enables you to connect a web app or website to Exhibitera Hub and include it in your exhibitions.

If you're not using Exhibitera Hub, Custom App setup might not be the easiest option. Instead, consider setting up Kiosk Mode in a browser of your choice.

To set up a custom app, you will need one of:

  • The source files for your web app;
  • A URL to the app or website.
Definition name must not be blank.
