Welcome to Image Compare!

Image Compare creates one or more image sliders, allowing a user to swipe back and forth between two images for a before/after effect. A single Image Compare instance can have up to eight pairs of images for users to select from.

To create an Image Compare, you will need:

  • At least one pair of image files;
  • Basic details like a title and subtitle for the app.
Definition name must not be blank.

What languages will you include?

Choose your image pairs

Image Compare can include up to eight pairs of images. Add those that you would like to include:

You must add at least one image pair.
Make sure you have selected images for each image pair.

Add image pair details

You can add short labels to each of your images, as well as a longer description that will be visible in a pop-up. You'll do this for each language you are supporting. You can always change these later.

Give your app a title

Visitors will first see a grid of icons representing your image pairs. On this "home screen," you can add a title and a subtitle to provide come context for what they will see.
