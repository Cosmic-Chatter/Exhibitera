Create a new definition

Welcome to Timelapse Viewer!

Timelapse Viewer lets you turn a video or image sequence into a touchscreen interactive.

To create a Timelapse Viewer, you will need:

  • The video file or images you want to visualize.
  • The title text that should be displayed.
Definition name must not be blank.

What files will you use?

Timelapse Viewer uses an image sequence. If you have a video, first convert it to images. Then, select the first and last image file in your sequence.

You must select both the first and the last file in your sequence.
The first and last files cannot be the same.

Configure the attractor

When no one is interacting with the timelapse, it will loop through your image sequence automatically. You have the option to display a short text header, such as "See a Flower Bloom", on top of the amimation.

How should the word cloud look?

You will be able to adjust colors and fonts after completing Guided Setup.

Convert a video

Timelapse Viewer works with a series of images. If you have a video, use this tool to convert it into an image sequence.
